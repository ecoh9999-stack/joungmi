🎲 로또 번호 생성기란?

로또 6/45 번호를 무작위로 생성해주는 무료 도구입니다. 1부터 45까지의 숫자 중 6개를 선택하며, 포함/제외 번호 설정, 번호별 통계 분석 기능을 제공합니다.

🎯 주요 기능

  • 자동 번호 생성: 1-45 중 랜덤 6개 번호 추출
  • 포함 번호 설정: 반드시 포함할 번호 지정
  • 제외 번호 설정: 제외할 번호 지정
  • 번호별 통계: 각 번호의 출현 빈도 분석
  • 색상 구분: 번호 범위별 색상으로 시각화
